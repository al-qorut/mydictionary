Kami menghargai privasi Anda dan berkomitmen untuk melindungi informasi pribadi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami menangani informasi yang mungkin kami kumpulkan melalui penggunaan aplikasi My Dictionary, yang dapat diunduh di Google Play Store.

  1. Informasi yang Kami Kumpulkan

    2. Aplikasi My Dictionary tidak mengumpulkan data apapun untuk bisa menggunakan seluruh fungsi aplikasi, email device-id atau apapun itu yang menyangkut data pengguna aplikasi.


  2. Izin Perangkat

    3. Aplikasi ini tidak meminta izin perangkat apa pun, kecuali untuk penggunaan penuh aplikasi standar yang tidak memerlukan izin khusus.


  3. Penggunaan Data

    4. Karena tidak ada pengumpulan data pengguna maka aplikasi tidak memerlukan data apapun dari pengguna untuk menjalankan penuh aplikasi


  4. Keamanan

    5. Aplikasi menjaga keamanan dan privasi data pengguna dengan tidak mengumpulkan data pengguna dan tidak ada kebutuhan data pengguna untuk menjalankan penuh seluruh aplikasi.


  5. Pihak Ketiga

    6. Aplikasi ini menggunakan data dari pihak ketiga, yaitu data-data kode terbuka yang ada di internet.
